## 🚀 如何使用这些文档

### 对于第一次访问的用户

1. 选择你的语言
   - 中文: [README.md](./README.md)
   - English: [README_EN.md](./README_EN.md)

2. 快速浏览 5 分钟

3. 选择你的角色，深入学习
   - 中文: [INDEX.md](./INDEX.md) 中的"按角色分类"
   - English: [INDEX_EN.md](./INDEX_EN.md) 中的"Learning Path by Role"

### 对于想要贡献的用户

1. 阅读贡献指南
   - 中文: [CONTRIBUTING_CN.md](./CONTRIBUTING_CN.md)
   - English: [CONTRIBUTING_EN.md](./CONTRIBUTING_EN.md)

2. 选择工作类型（功能/Bug 修复/文档/翻译）

3. 按照工作流步骤进行

### 对于想要翻译的用户

1. 阅读国际化指南
   - [INTERNATIONALIZATION.md](./INTERNATIONALIZATION.md)
   - [INTERNATIONALIZATION_CN.md](./INTERNATIONALIZATION_CN.md)

2. 选择要翻译的文档（优先级：README → CONTRIBUTING → AGENTS）

3. 按照翻译工作流提交 PR
