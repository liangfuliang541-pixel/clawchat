## 🎓 使用建议

### 对于 GitHub 访客

1. ⭐ Star 项目
2. 📖 阅读你语言的 README
3. 🔗 点击"查看更多"进入完整文档
4. 💬 加入讨论区

### 对于想要部署的用户

1. 选择语言 (README)
2. 查看快速开始
3. 前往 DEPLOYMENT_GUIDE
4. 按步骤部署

### 对于想要贡献的开发者

1. Fork 项目
2. 阅读 CONTRIBUTING 指南
3. 选择任务 (Issue)
4. 按照工作流提交 PR

### 对于想要翻译的志愿者

1. 阅读 INTERNATIONALIZATION 指南
2. 选择要翻译的文档
3. 在 Discussions 中声明
4. 提交翻译 PR
