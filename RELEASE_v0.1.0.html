🚀 快速开始

1. 克隆仓库

git clone https://github.com/YOUR_USERNAME/clawchat.git

2. 安装依赖

npm install --legacy-peer-deps

3. 开发模式 (需要MongoDB)

npm run dev

4. Docker部署

docker-compose up -d

5. 运行测试

npm run test -w backend
